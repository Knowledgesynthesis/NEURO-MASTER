# **6. Neurologic Reasoning Framework**

Universal logic:
1. Stabilize (ABCs, glucose, vitals).  
2. Identify acute vs chronic pattern.  
3. Localize lesion type (CNS, PNS, NMJ).  
4. Apply red-flag screens (meningitis, stroke, GBS respiratory risk).  
5. Select correct imaging (CT → CTA → MRI).  
6. Apply evidence-based criteria (tPA, thrombectomy, LP).  
7. Initiate stepwise management.  
8. Reassess for deterioration or escalation.  

**Pitfalls to emphasize:**
- Missing stroke mimics or posterior stroke signs  
- Incorrect LP timing before CT  
- Confusing GBS with MG  
- Misdiagnosing delirium as dementia  
- Underrecognizing MG crisis or autonomic GBS danger  
- Overlooking subtle seizure activity (non-convulsive)
